### Installation

CLONE REPO

```bash

> git clone https://github.com/ghost717/wp-adminLTE-api project

```

INSTALL DEPEDENCIES

```bash

> cd project

> composer install

> npm install

```

CONFIG & INSTALL DB

```bash

> cp -r .env.example .env

> php artisan migrate:fresh --seed

> php artisan key:generate

```
